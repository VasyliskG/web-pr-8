# web-pr-8
Практична робота №8

Тема: Створення інтерактивних елементів вебсторінки за допомогою
чистого JavaScript (без бібліотек).
Мета: Навчитись реалізовувати основні інтерактивні елементи на
сторінці: випадаюче меню, динамічну зміну кольору елементів, а також показ
і приховування частини контенту.

Короткі теоретичні відомості

Все що потрібно, викладено в лекції №7.
Завдання

1. Реалізувати випадаюче меню зі стрілочкою, яке з’являється при
натисканні на кнопку.
2. Реалізувати зміну кольору блоку при кожному кліку.
3. Створити блок тексту, в якому спочатку видно лише три рядки,
а при натисканні на кнопку — розкривається весь текст, або ховається назад.
4. Створити кнопку, яка підраховує кількість натискань і
відображає поточне число.
5. Створити блок, в якому після натискання кнопки буде
виводитись поточна дата і час у форматі дд.мм.рррр.
6. Оформити звіт (додаток 2).
7. Проект закинути в класрум.

Додатково (за бажанням)

 Створити список і поле введення. При введенні тексту в поле —
автоматично фільтрувати список, показуючи тільки ті елементи, які містять
введене слово.
 Створити два поля введення та кнопку. При натисканні на кнопку
обчислити суму двох чисел і вивести результат на екран.
